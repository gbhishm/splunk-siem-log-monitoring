## Project Overview

This project demonstrates the implementation of a Splunk SIEM lab for monitoring Windows security events and performing security log analysis.

The project focuses on detecting authentication events, failed login attempts, suspicious activities, and analyzing security logs using Splunk Search Processing Language (SPL).
